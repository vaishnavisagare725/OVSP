# Online Voting System

## Project Description
Online Voting System is a web-based application developed using .NET Framework, C#, and Visual Studio 2019.
The project is designed to conduct voting online in a secure and efficient manner.

This system allows users to cast their votes digitally, while the admin manages elections, candidates, and results.

## Technologies Used
- .NET Framework
- C#
- ASP.NET
- SQL Server
- HTML
- CSS
- Visual Studio 2019

## Modules

### Admin Module
- Admin login
- Add and manage candidates
- Manage voters
- View election results

### User (Voter) Module
- User registration and login
- Cast vote online
- View voting status

## Features
- Secure login system
- One voter can vote only once
- Fast and accurate vote counting
- User-friendly web interface

## How to Run the Project
1. Open the project in Visual Studio 2019
2. Configure the SQL Server database
3. Update the connection string in web.config
4. Run the project using IIS Express

## Purpose of Project
This project is developed for academic and learning purposes to understand:
- Web-based application development
- Backend programming using C#
- Database connectivity using SQL Server
